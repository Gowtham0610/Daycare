# Family-Friendly Day Care Website

A pixel-perfect frontend implementation of a daycare website based on a Figma design. This project focuses on accurately translating design into code, maintaining exact typography, layout, spacing, and visual styles.

---

## Live Demo
https://loveandcare.org.in/

---

## Features

- Pixel-perfect UI matching the original Figma design
- Responsive layout for mobile, tablet, and desktop
- Exact fonts, colors, gradients, and spacing
- Smooth transitions and basic animations

---

## 🛠️ Tech Stack

- React (Vite)
- CSS / Tailwind CSS
- Framer Motion (for animations)
